# Changelog

## 0.1.0 (Unreleased)

- Initial VS Code extension: **Open Code Dark** color theme, optional **inline completions** (local Gemma if available), **Autodrive** webview, **Run agent** with built-in **diff** review and accept/reject, **clear/show memory** commands.
- **Discriminator** on prompts/edits; **Run discriminator on commit message** when editing a commit buffer.
- Rust `open-code-memory` + `open-code-memoryd`: two-layer **SQLite** (raw + semantic), HTTP on localhost, `Cargo.lock` and **Rust 1.85** toolchain.
- **CI** (fmt, clippy, test, build) + **fork-health** workflow preserved.
- Docs: **BUILD**, **GOVERNANCE**, **RELEASE**, **SECURITY**; **MVP_AND_ROADMAP** (full plan, gaps, suggestions); **VSIX** via `npm run package` in workspace.
